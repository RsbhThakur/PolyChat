Setup Instructions before running this application:
    1. Make sure npm and nodejs are installed in your enviornment.
    2. Open terminal in the backend folder of the Project.
    3. Run "npm install"
    4. Rename the 0.env to .env
    5. Place your own API keys and MONGODB connection URI through the links available in the .env file.
    6. Run "npm run dev" to start the server.
    7. Open "127.0.0.1:3000" or "localhost:3000" in your browser to use the application.
